# Work Attendance iTab（Edge 小标签版）

这是一个 Edge 工具栏小标签（popup）扩展，不是整页新标签页。

## 你要的形态

- 点击浏览器右上角扩展图标，弹出一个小面板
- 在小面板里完成打卡、收入录入、导入导出

## 功能

- 月历打卡（单击先选中，再次单击切换打卡）
- 默认日薪 + 当日收入预填（未打卡也可预填）
- 本月统计：已打卡、预计收入、预填收入
- 导入/导出 JSON

## 安装

1. 打开 `edge://extensions/`
2. 开启“开发者模式”
3. 点击“加载解压缩的扩展”
4. 选择你解压后的扩展目录（包含 manifest.json 的文件夹）
5. 建议把扩展“固定到工具栏”

## 文件

- `manifest.json`：扩展配置（已改为 `action.default_popup`）
- `popup.html`：小标签页面
- `popup.css`：小标签样式
- `popup.js`：小标签逻辑
