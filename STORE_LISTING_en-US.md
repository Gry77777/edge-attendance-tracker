# Store Listing (English)

## Name
Work Attendance iTab

## Short Description
A compact Edge toolbar popup for attendance check-in and income tracking.

## Full Description
Work Attendance iTab is a lightweight attendance extension for Edge. Click the toolbar icon to quickly check in and track your daily/monthly income.

Highlights:
- Month/Week view toggle
- Streak badge with break reminder
- Default daily wage + per-day income override
- Progress and income visualization
- Three themes (Mist / Ocean / Paper)
- Quick mode for ultra-fast check-in
- Local JSON import/export backup

Privacy:
- All data is stored locally in browser storage.
- No automatic upload to any remote server.
