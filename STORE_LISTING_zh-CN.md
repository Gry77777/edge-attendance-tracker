# 商店文案（中文）

## 名称
Work Attendance iTab

## 一句话简介
在 Edge 工具栏小弹窗中快速完成打卡、收入记录与月度统计。

## 详细介绍
Work Attendance iTab 是一个轻量的工作打卡扩展，打开浏览器右上角图标即可记录当天出勤与收入。

功能亮点：
- 月视图 / 周视图切换
- 连续打卡天数与断签提醒
- 默认日薪 + 单日收入预填
- 预计收入、预填收入、进度条可视化
- 三套主题（晨雾 / 海盐 / 纸感）
- 极速模式（紧凑一屏，适合秒打卡）
- 本地 JSON 导入/导出备份

隐私说明：
- 数据仅保存在本地浏览器，不上传云端。
